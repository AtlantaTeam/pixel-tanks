## Последние принятые решения

- **AFK-вердикт для Димы**: внутри фазы — полностью AFK (issues → PR → ревью);
  между фазами HITL остаётся одна кнопка — мердж PR. Авто-мердж НЕ делаем:
  ревью Фазы 2 нашло HIGH (resize терял детерминизм террейна) — авто-мердж
  пропустил бы его в main. Фазы зависимы (3 строится на 2), потому мердж
  нужен до старта следующей фазы.
- Ручные гейты (npm install на ask — осознанно оставлен): агент вешает на issue
  label `blocked` + комментарий, раннер такие пропускает. Label снимает человек.
- Milestone закрывается свипом раннера, когда issues разобраны И PR фазы
  смерджен (не раньше — ревью может вернуть работу).
- Ревьюер не мог постить в PR — `gh api repos/...` добавлен в allow
  (settings.json). Ревью Фазы 2 выложено супервизором вручную.
- Сквош-мердж фаз: ветка после мерджа расходится с main — новые инфра-коммиты
  возить отдельной веткой от main (как PR #42), не пушить в смерженную.
- E2e: `@playwright/test` установлен; NO_PROXY для localhost зашит в
  playwright.config.ts (локальный VPN-прокси 127.0.0.1:10810 ронял пробу
  webServer 502). Зомби next dev на 3050 убивать через netstat/Stop-Process.
- Роутинг моделей работает (проверено): medium→sonnet, high→opus,
  expert→fable (#9 Фазы 3 пойдёт на fable).
