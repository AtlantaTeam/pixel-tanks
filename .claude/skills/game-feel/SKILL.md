# Game feel: чек-лист сочности

Механика «работает» ≠ механика «ощущается». После каждой игровой фичи — прогон по чек-листу. Не всё обязательно внедрять сразу — но каждое «нет» должно быть осознанным решением, а не забытым.

## Выстрел

- [ ] Отдача ствола/корпуса (2–3 px назад, 100 мс)
- [ ] Вспышка на дуле (1–2 кадра)
- [ ] Звук выстрела (Web Audio, с лёгкой рандомизацией pitch ±10%)

## Полёт снаряда

- [ ] Трейл/след за снарядом (частицы или fade-линия)
- [ ] Снаряд визуально реагирует на ветер (наклон/дрожание)

## Попадание / взрыв

- [ ] Частицы взрыва (земля, искры) — с пулом объектов
- [ ] Screen shake (амплитуда от урона, 150–300 мс, затухание)
- [ ] Кратер в террейне с рваным краем, не идеальный круг
- [ ] Звук взрыва (отличается для попадания в танк / в землю)
- [ ] Флэш урона на танке + всплывающие цифры урона

## UI-фидбек

- [ ] Кнопки: hover/active-состояния, мгновенный отклик (<100 мс)
- [ ] Смена хода — явная анимация/индикация, не мгновенный телепорт камеры
- [ ] HP-бар анимируется к новому значению, не прыгает
- [ ] Победа — пауза перед диалогом (дать увидеть последний взрыв)

## Проверка

Прогнать скилл `playtest`, смотреть скриншоты анимационных состояний. Отчёт: таблица чек-листа (есть/нет/осознанно пропущено).
