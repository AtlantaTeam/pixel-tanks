# Read (токен-эффективно)

Прочитай $ARGUMENTS эффективно:

1. Сначала структура: `head -50 {file}` или Grep по `function|class|export|model`.
2. Найди нужную часть: `grep -n "{имя}" {file}`.
3. Читай только нужные строки (Read с offset/limit или `sed -n 'N,Mp'`).
4. JSON: `jq '.нужное_поле' {file}` вместо чтения целиком.
5. Prisma-схема: только нужную модель — `sed -n '/model {Name}/,/^}/p' schema.prisma`.

Никогда не читай файл целиком, если нужна одна функция или модель.
