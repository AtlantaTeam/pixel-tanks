# PRD: game-next — некст-ген апгрейд игры

**Дата**: 2026-07-18
**Статус**: Draft

## Цель

Превратить портированную игру 2021 года в современную «сочную» аркаду: игра должна ощущаться живой (juice, характер бота) и давать повод возвращаться и делиться ею (daily challenge, реплеи-ссылки). Rewrite ради rewrite не продаёт — нужен геймплейный скачок.

## Пользовательские сценарии

### A. Juice / game feel

- Игрок заходит в меню / бой → играет оригинальная музыка из старой игры (`game-menu.mp3` / `gameplay.mp3`, добавлена Андреем), есть кнопка mute
- Игрок стреляет → снаряд летит с трейлом, слышен выстрел (`fire.wav`)
- Снаряд попадает в землю → взрыв с частицами земли, screen shake, звук взрыва
- Снаряд попадает в танк → slow-mo момент попадания, усиленный shake, вспышка урона
- Игрок промахивается / попадает → счёт и ходы обновляются с анимацией, не мгновенным скачком

### B. Характер бота (Терминатор 2.0)

- Бот попадает в игрока → над танком бота пиксельный chat-bubble с довольной репликой («Hasta la vista»)
- Игрок попадает в бота → bubble со злой репликой («Эй, полегче!»)
- Игрок стреляет в себя / нелепо промахивается → саркастичная реплика
- Реплики берутся из пула старой игры (`bot-messages`), браузерные Notifications не используются

### C. Daily challenge + реплеи

- Игрок открывает «Бой дня» → у всех игроков в эти сутки один и тот же seed террейна/ветра
- Игрок завершает бой дня → результат уходит в лидерборд дня (Payload, фаза 7)
- Игрок жмёт «Поделиться» → получает короткую ссылку (seed + последовательность ходов)
- Друг открывает ссылку → смотрит покадровый реплей боя без регистрации

### D. Мобилка (приоритет)

- Игрок открывает игру на телефоне (портрет или ландшафт) → игра занимает весь экран, без горизонтального скролла, HUD не перекрыт системными панелями
- Игрок тянет пальцем от танка и отпускает → направление задаёт угол, длина оттяжки — мощность, отпускание — выстрел (как в Angry Birds); при оттяжке видна превью-линия прицела
- Игрок играет на среднем телефоне → стабильный FPS, эффекты деградируют (меньше частиц), а не тормозят игру

## В скоупе

1. **Мобилка**: тач-управление «оттяни и отпусти» с превью прицела, адаптивный HUD (портрет + ландшафт), safe-area, тач-цели ≥ 44px.
2. **Клавиатура и мышь (desktop)**: полноценная клавиатурная схема как в старой игре — стрелки: точная настройка угла/мощности, Ctrl+стрелки: смена оружия / перемещение танка, Enter/Space: выстрел; мышь — угол/колесо/клик. Три схемы (тач / клавиатура / мышь) равноправны.
3. **Juice-пакет**: screen shake, частицы взрыва, трейл снаряда, slow-mo попадания, анимация HUD. Проверка скиллом `game-feel`.
4. **Звук — возврат оригинала**: аудио из `../pocket-tanks/static/audio/` (музыка меню и боя — мелодии Андрея, эффекты выстрела и двух взрывов) через WebAudio; фоновая музыка + mute-переключатель.
5. **Бот-реплики**: перенос пула реплик из `../pocket-tanks/src/modules/bot-messages/`, рендер chat-bubble на Canvas/DOM поверх танка, привязка к игровым событиям (попадание/промах/самострел).
6. **Daily challenge**: генерация дневного seed, режим «Бой дня» рядом с обычным, запись результата.
7. **Реплеи**: сериализация боя в компактную строку (seed + ходы), страница воспроизведения по ссылке.

## Не в скоупе

- Реалтайм-мультиплеер (WebSocket/WebRTC) — отдельная дорогая ставка, бэклог
- Асинхронный PvP «по переписке» — бэклог, ляжет поверх механики реплеев
- Нативная обёртка / публикация в сторы (PWA-манифест — можно, TWA/Capacitor — нет)
- Мета-прогрессия и скины танков (в т.ч. генерация артов через OpenAI API) — бэклог
- LLM-генерация реплик бота — сначала статичный пул, LLM — следующая итерация
- Forum и Privacy из старого проекта — выкидываем окончательно
- Profile — не часть game-next, закрывается в фазе auth

## Технические ограничения

- Физика детерминирована, random только через инжектируемый seed (`.claude/rules/canvas.md`) — реплеи и daily challenge опираются на это; любой недетерминизм ломает фичу C
- Никаких аллокаций в кадре: частицы — через пул объектов
- React вне кадрового цикла: juice-эффекты живут в движке, в Zustand — только события
- Лидерборд дня зависит от фаз 5–7 (Payload users/scores); сама механика seed и реплеев — нет, можно делать до auth
- Звук — только после первого пользовательского жеста (autoplay policy браузеров)
- Генерация картинок (когда дойдём до скинов): OpenAI API (`gpt-image-1`), аккаунт у пользователя есть; ключ — только из env, оффлайн-генерация ассетов, не в рантайме игры
- Слабые устройства: у juice-эффектов должен быть деградационный путь (меньше частиц), FPS не ниже 60 на desktop и не ниже 30 на среднем телефоне
- Мобильный вьюпорт: `h-dvh` (не `100vh`), safe-area insets, `touch-action` на Canvas против случайного скролла/зума, тач-цели ≥ 44px
- Canvas учитывает `devicePixelRatio` и resize (ResizeObserver) — критично для ретины телефонов

## Критерии готовности

- [ ] В меню и бою играет оригинальная музыка из старой игры, mute-переключатель работает и переживает перезагрузку страницы
- [ ] Взрыв сопровождается частицами, screen shake и звуком; попадание в танк — slow-mo (проверка скиллом `game-feel` + `playtest`)
- [ ] FPS в бою не падает ниже 60 при всех эффектах (замер в `playtest`)
- [ ] Бот показывает реплику в chat-bubble на каждое событие попадания/промаха; пул реплик ≥ 15 (весь старый набор)
- [ ] Два игрока, открывшие «Бой дня» в одни сутки, получают идентичный террейн и ветер
- [ ] Ссылка реплея открывает покадровое воспроизведение боя, идентичное оригиналу (тест: seed + ходы → одинаковый итоговый счёт)
- [ ] Физика с фиксированным seed покрыта детерминированными unit-тестами (вход → траектория)
- [ ] Все эффекты уважают `prefers-reduced-motion` (shake/slow-mo отключаемы)
- [ ] Полный бой с ботом играбелен только клавиатурой, без мыши (стрелки, Ctrl+стрелки, Enter/Space)
- [ ] На вьюпорте 375px (портрет и ландшафт) игра полностью играбельна жестом «оттяни и отпусти»: полный бой с ботом от старта до победы (Playwright, touch-эмуляция)
- [ ] Нет горизонтального скролла, HUD виден целиком, все тач-цели ≥ 44px на 375px-вьюпорте
